# 🧑‍⚖️ Agente Legal Procesal IA – México

**Agente IA especializado en Derecho Procesal Mexicano**  
Actualizado al **06 de mayo de 2026** con el marco jurídico vigente.

### ¿Qué puede hacer este agente?
- Analizar casos completos (hechos, normas y pruebas)
- Diseñar estrategias judiciales ganadoras
- Preparar escritos, alegatos, recursos y demandas
- Simular audiencias, interrogatorios y refutaciones
- Negociar acuerdos estratégicos (conciliación, acuerdos reparatorios, transacciones)
- Aplicar el CNPCF, CNPP y Reforma Judicial 2024 con precisión absoluta

### Marco jurídico vigente (mayo 2026)
- Código Nacional de Procedimientos Civiles y Familiares (CNPCF) – reforma 15-ene-2026
- Código Nacional de Procedimientos Penales (CNPP) – reforma noviembre 2025
- Ley de Amparo (reformas 2025)
- Ley Orgánica del Poder Judicial de la Federación
- Reforma Judicial 2024 (elección popular de jueces y nueva estructura del PJF)

### Estructura del repositorio
- `SKILLS.md` → Habilidades core del agente (ya creado)
- `docs/` → Documentación técnica y marco jurídico
- `prompts/` → Prompts listos para usar en Grok, Claude, ChatGPT, etc.
- `knowledge-base/` → Base de conocimiento (para futuro RAG)
- `tests/` → Casos de prueba
